# Security Policy

Scope
- This repository hosts documentation and schemas only.
- Report security issues that could mislead implementers or weaken
  cryptographic, identity, or privacy requirements.

Reporting
- Email: security@arky.foundation
- Include: affected spec/schema, description, impact, reproduction, references.
- We acknowledge within 5 business days and provide a resolution plan.

Handling
- We track issues privately until a coordinated fix is ready.
- Changes appear as spec errata, schema updates, or registry advisories.

No secrets
- Do not include secrets, PII, or production material in examples.
